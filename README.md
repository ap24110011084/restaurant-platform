# Restaurant Reservation & Management System

A full-stack MERN web application for restaurant booking and management.

## Features
- User Login & Registration
- Table Reservation
- Waitlist Management
- Admin Dashboard
- Manage Users
- Manage Tables
- Reservation Tracking
- Responsive UI

## Tech Stack
Frontend: React, Vite, Tailwind CSS  
Backend: Node.js, Express.js  
Database: MongoDB

## Folder Structure
- backend/
- frontend/

## Run Project

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
